# 🛣️ AI-Based Pothole Detection using YOLOv12

A Streamlit web application for detecting potholes in road images and videos using a custom-trained **YOLOv12** model.

### 🌐 Live Demo

https://pothole-computer-vision-project-nv7nw8jqhwcfdnv2vojsjt.streamlit.app/

### 🚀 Features

* Detect potholes from images and videos
* Adjustable confidence threshold
* Detection analytics (confidence, size, severity)
* Bounding box visualization
* Download processed images and videos
* Interactive Streamlit interface

### 🛠️ Tech Stack

* Python
* YOLOv12
* Ultralytics
* OpenCV
* Streamlit
* NumPy
* Pillow

### 📦 Run Locally

```bash
git clone https://github.com/SHIVANG577/Pothole-Computer-Vision-Project.git

cd Pothole-Computer-Vision-Project

pip install -r requirements.txt

streamlit run app.py
```
